继承弊端：破环了封装，会覆盖父类当中的方法




final关键字  “最终的，不可修改的”

可以保证数据的安全性

private 只在当前类子类访问不到

final：子类可以使用但是不能修改

protected  在同一个包同一个类内可以同时访问


字段：可以使用
特征：不能在修改字段

方法：可以使用
特征：子类不能在修改，或者说不能再覆盖他的内容。

类：可以使用
特征：不能再继承



细节：

修饰字段时：默认没有初始值，必须手动设置初始值


final修饰变量时：代表他为一个常量，命名规则，修饰时所有字母必须大写。

final可以在局部代码块使用

修饰基本数据类型  代表值不能再修改了。
修改引用数据类型  代表地址不能再修改了（new）对象引用数据类型。除了String以外。
