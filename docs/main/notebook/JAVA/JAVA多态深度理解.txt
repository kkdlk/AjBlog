狗是动物类

子类是父类的一种特殊形式。

多态的概念：一个对象有多种形态
	    既可以用自己的类表示
	    也可以用他的父类表示。

多态的特点：Animal dog = new Dog();
	    Dog dog = new Dog();
	   把子类对象赋给父类变量
	   在运行时期会表现出具体的子类特征调用子类的方法。
	   运行时表现得是子类的特征
先在new出来的子类寻找，找不到会在寻找父类的。

getClass 获取当前对象的真实类型
